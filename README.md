Ціль проекта порахувати нагрузку робробника і тестувальника на спрінт, по даним Jira.
Фронт Ангуляр, апі NestJS 

Налаштування. Необхідно скопіювати файли і заповнити параметри
    1. front/src/app/params.ts-dist в front/src/app/params.ts
    2. jira-api/server/.env в jira-api/server/.env.local

Запуск через докер:
docker-compose up
